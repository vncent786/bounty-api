## Description
Bounty API provides verified Singapore property, tax, affordability, transaction, and location data for AI agents via MCP and x402 micropayments. Singapore is live now; the API shape includes a region parameter for future market expansion.

## Current MCP Tools
- sg_stamp_duty — Singapore BSD + ABSD calculator
- sg_postal_lookup — Singapore postal code to district lookup
- sg_address_intel — postal code to district, planning area, market region, nearest MRT
- sg_mrt_near — nearest MRT stations to a postal code
- sg_mrt_search — MRT station search
- sg_affordability — MAS TDSR/MSR mortgage affordability calculator
- sg_property_analyze — complete property investment analysis
- sg_property_rank — rank candidate properties by value, yield, affordability, and location
- sg_rental_yield — rental yield calculator
- hdb_resale_median — HDB resale medians by town/flat type
- hdb_resale_search — HDB resale transaction search

## Categories
Finance, Real Estate, Data, Singapore, x402, Payments, Government Data, MCP
